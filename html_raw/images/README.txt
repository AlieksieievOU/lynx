This folder contains the exported SVGs and mapping for PNG images used in the LYNX Permits Inc. static HTML export.

SVG Files:
- mission-star.svg (Our Mission section)
- message-icon.svg (Reach Out section message bubble)
- social-fb.svg, social-ig.svg, social-li.svg (Social icons)
- location-pin.svg, phone-icon.svg (Contact and Footer icons)

PNG Image Mapping:
To use these in a standard web server, manually export these assets from Figma and rename them as follows:

1. logo-dark.png (Main logo for white backgrounds)
2. logo-light.png (Main logo for dark backgrounds/footers)
3. hero-truck.png (Large red truck image in Hero section)
4. driver-with-truck.png (Image in Permitting Procedures section)
5. specialist.png (Image of permit specialist in Services hero)
6. client-atc.png (Avatar/Logo for ATC Transport testimonial)
7. service-permit.png (Thumbnail for Permit Acquisition service)
8. service-logistics.png (Thumbnail for Logistics Management service)
9. service-compliance.png (Thumbnail for Compliance Consulting service)
10. mission-bg.png (Background image for Our Mission section)
11. location-map.png (Google Maps image in Discover section)
12. stamping-docs.png (Image for Permitting Procedures in Services page)

All HTML files have been updated to reference these images as 'images/[filename].png'.
